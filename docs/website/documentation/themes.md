# Custom Themes

Custom themes let you adapt ahaCode to your workflow and visual preferences. You can change both the app UI colors and the editor syntax highlighting colors.

## How it works

- Themes are stored in `~/.ahacode/themes/` as JSON files.
- Both light and dark theme types are supported.
- Theme files are watched in real time — edit the JSON, see the result instantly.

## Creating a Theme

Create a new theme from **Settings → Themes**. ahaCode generates a Rose Pine-based JSON template that you can use as a starting point and edit in real time.
