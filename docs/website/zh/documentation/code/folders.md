# 文件夹

文件夹帮助你按项目、主题、语言或任何适合你工作流的结构来组织代码片段。当你在侧边栏中选择一个文件夹时，ahaCode 会在第二列中显示该文件夹及其所有子文件夹中的代码片段。

## 添加文件夹

使用以下方法之一：

- 在菜单栏选择 **"文件"** > **"新建文件夹"**。
- 点击资料库中 **"文件夹"** 旁边的 **"+"** 按钮。
- 按 <kbd>Shift+Cmd+N</kbd>（macOS）或 <kbd>Shift+Ctrl+N</kbd>（Windows/Linux）。

## 移动文件夹

将一个文件夹拖放到另一个文件夹上，即可创建嵌套文件夹结构。

## 重命名和删除

右键点击文件夹，选择 **"重命名"** 或 **"删除"**。

::: warning
删除文件夹会将其所有代码片段和子文件夹移至 **"收件箱"**。
:::

## 设置默认文件夹语言

当文件夹中大多数代码片段使用相同语法时，可以设置默认语言。

右键点击文件夹，选择 **"默认语言"**。

## 设置自定义图标

自定义图标有助于大型资料库更易于扫视。

右键点击文件夹，选择 **"设置自定义图标"**。

::: info
要恢复默认图标，在图标选择对话框中点击 **重置** 按钮。
:::
