# 片段

片段是代码片段内的标签页。当一个代码片段需要多个相关的版本、文件或语言时，可以使用片段将它们集中在一起。

## 添加片段

使用以下方法之一：

- 在菜单栏选择 **"文件"** > **"新建片段"**。
- 点击编辑器右上角的 **"+"** 按钮。
- 按 <kbd>Cmd+T</kbd>（macOS）或 <kbd>Ctrl+T</kbd>（Windows/Linux）。

## 重命名和删除

右键点击片段标签，选择 **"重命名"** 或 **"删除"**。

::: warning
删除片段会将其永久移除。
:::
