# 资料库

资料库是 Code 空间的第一列，帮助你快速浏览系统文件夹并跳转到代码片段集合的任意部分。

## 收件箱

收件箱是尚未整理到文件夹中的代码片段的默认存放位置。

## 收藏夹

收藏夹用于存放你想要从资料库任何位置快速访问的代码片段。

## 全部片段

全部片段在一个页面中展示你的整个代码片段集合。

## 回收站

删除的代码片段会被移至回收站。要永久删除，右键点击 **"回收站"** 并选择 **"清空回收站"**。

::: warning
清空回收站会永久删除其中的代码片段，此操作无法撤销。
:::
