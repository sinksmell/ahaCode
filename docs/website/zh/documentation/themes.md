# 自定义主题

自定义主题让你可以根据工作流和视觉偏好来调整 ahaCode。你可以同时更改应用 UI 颜色和编辑器语法高亮颜色。

## 工作原理

- 主题以 JSON 文件的形式存储在 `~/.ahacode/themes/` 目录中。
- 支持浅色和深色两种主题类型。
- 主题文件被实时监控——编辑 JSON 即可立即看到效果。

## 创建主题

在 **设置 → 主题** 中创建新主题。ahaCode 会生成一个基于 Rose Pine 的 JSON 模板，你可以将其作为起点并实时编辑。
